Learn more about the Arena!

I've been working on this game in my spare time for the past year. I made the prototype in C++, and am in the process of porting it over to Unreal Engine and fleshing it out into a full-fledged game! Check out the history page and the Learn section to find out more about the game!

A Brief History

The Arena started out in raw C++, run through text input in the console. Each symbol was a different unit, and underscore's were empty spaces.

When Unreal Engine went 100% free in March, I decided to port my game over! However, I'd never used a game engine before. I started learning my way around by making a dynamically generated chess game.

Within a few weeks, I had a working unit and board system implemented. Everything was pretty rough-hewn, but Unreal Engine made it look pretty anyways.

Over the past few months, I've focused on implementing more of my units in Unreal and refining the UI.

Check out the kickstarter page for more progress pictures & information!
